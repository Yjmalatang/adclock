# Ad Clock — 投放时区助手

出海信息流投放的时区转换工具，帮助投放团队快速计算全球各地投放时段与北京时间的对应关系。

## 核心功能

- **时段反推**：设定目标投放时段（当地时间），一键查看对应北京时间，支持多地区同时对比
- **世界时钟**：实时显示多个投放市场的当前时间
- **零点对照**：各国 0:00 对应北京几点，方便设置广告日预算重置时间
- 覆盖 40+ 投放市场：美国四时区、东南亚、中东、欧洲、南美、非洲
