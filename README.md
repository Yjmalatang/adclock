# Ad Clock — 投放时区助手

信息流投放时区转换工具，帮助出海投放团队快速计算全球各地投放时段与北京时间的对应关系。

**在线使用** → [https://takeoutnft.github.io/adclock/](https://takeoutnft.github.io/adclock/)

## 功能

- **时段反推**：设定目标投放时段（当地时间），一键查看对应北京时间，支持多地区同时对比
- **世界时钟**：实时显示多个投放市场的当前时间
- **零点对照**：各国 0:00 对应北京几点，方便设置广告日预算重置时间

## 内置时区

覆盖 40+ 投放市场：美国四时区、东南亚、中东、欧洲、南美、非洲等主要广告市场。

## 使用方式

- 浏览器直接访问，支持 PWA 安装到手机桌面
- 纯前端，无需后端，数据保存在本地

## Tech Stack

`HTML` `CSS` `JavaScript` `PWA` `Service Worker`
